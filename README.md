# Vue + Django 项目

## 后端

```powershell
.\.venv\Scripts\Activate.ps1
python .\backend\manage.py runserver 127.0.0.1:8000
```

- 管理后台：`http://127.0.0.1:8000/admin/`
- 已安装 `django-simpleui`
- Django 语言：`zh-hans`
- Django 时区：`Asia/Shanghai`

## 前端

```powershell
cd frontend
npm run dev
```

- 前端地址：`http://127.0.0.1:5173/`
